🌍 wasm-spatial-core

High-performance WebAssembly spatial data processing engine for frontend Web3D/GIS

v0.2.0 Rust + WASM 456 KB gzipped Zero dependencies
💡 演示在浏览器内运行。首次打开需下载约 1MB WASM（Wi‑Fi 通常数秒，手机 4G 可能 15～40 秒）。
📝 Minimal Usage
// 1. Import the WASM module import init, { batchWgs84ToGcj02, parseGeoJsonCoords, SpatialIndex, version, } from 'wasm-spatial-core'; // 2. Initialize (load WASM binary) await init(); console.log(version()); // "0.2.0" // 3. Transform coordinates (WGS84 → GCJ02) const coords = new Float64Array([116.4, 39.9]); const result = batchWgs84ToGcj02(coords); // → Float64Array [116.416, 39.943]
